gServ

What is gServ?

gServ is a tool for creating and deploying REST based services without the hassle of a container (JBoss, Jetty, etc.) . Using gServ, you can easily define REST resources in Groovy.

Features:
  • Container Free
  • No other dependencies
  • Serve Groovy script as REST reources
  • Plugin Framework
  • Embeddable
  • Standalone Mode
  • CORS support
  • Compression support
  • ETag support
  • Basic Authentication
  • HTTPS Requirements:
  • Java JDK 1.6+

gServ can be used in two ways
  • Framework
  • Standalone
